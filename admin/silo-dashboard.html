Separation model (kept siloed)

  • Main campaign website remains public at arafatforcongress.org.
  • Silo runs under separate path/subdomain and keeps its own auth/token/data store.
  • Only summary bridge data is shared into the silo (public metrics/outreach/volunteer snapshot).

Improvement options review

  • Add role-based permissions (import-only, canvass-only, admin).
  • Add encrypted storage + backups for apps/voter-mapping-silo/data/store.json.
  • Add daily alert if volunteer sync is stale or import reject rate spikes.
